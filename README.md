# ua4iugi-pwa

The meaning of YAFIYGI is: You asked for it, you got it.
The meaning of PWA is: Progressive web application.

## `Template`
```
$ npx create-react-app ua4iugi-pwa --template typescript --use-npm
```

## `Setup`
```
$ git flow init -d
```

## `Build Scripts`
In the project directory, you can run:
```
$ npm start
$ npm test
$ npm run build
$ npm eject
```
